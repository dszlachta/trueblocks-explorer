Editable

Editable - an inline text editor with actions

Editable
Parameters
props (any)
editable (bool) if false, present the value without editing control
className (string) a display class name (defaults to 'editable_text')
fieldName (string) the name of the field being edited
fieldValue (string) the current value of the field being edited
record_id (string) a unique identifier for the record being edited (may be empty)
placeholder (string) text to display if the fieldValue is empty (defaults to fieldName)
isActive (string) starts the editor in edit mode
onValidate (string) called on blur event, should return true if input is value, error text otherwise (optional)
onChange (string) sends an action each time the text changes
onAccept (string) called after the input is validated
